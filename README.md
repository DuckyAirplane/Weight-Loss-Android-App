# Weight Loss Android App

Simple Android app to track weight entries.

## Features
- Login screen
- Weight log (RecyclerView)
- Add weight (dialog)
- Delete weight entries
- Settings screen (SMS reminders UI)

## Tech
- Java
- SQLite (DBHelper)
- Android Studio

## How to run
1. Open the project in Android Studio
2. Sync Gradle
3. Run on an emulator or Android device
